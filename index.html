About Facebook In-App Browser User Agents

Facebook's In-App Browser (a customized version of Chrome for Android) has specific patterns in its user agent string. A valid Facebook In-App Browser UA should contain:

  • [FBAN/ or [FB_IAB/ prefix indicating Facebook App
  • FBAV/ followed by the Facebook app version
  • Mobile device information
  • Android version
  • Chrome/ version
  • Other Facebook-specific identifiers

Example valid UA: Mozilla/5.0 (Linux; Android 10; SM-G960U Build/QP1A.190711.020; wv) AppleWebKit/537.36 (KHTML, like Gecko) Version/4.0 Chrome/89.0.4389.90 Mobile Safari/537.36 [FBAN/EMA;FBLC/en_US;FBAV/300.0.0.40.119;]
